# vize_ex_nif

Rust NIF for the [Vize](https://hex.pm/packages/vize) Elixir package.

Wraps [vize_atelier_sfc](https://crates.io/crates/vize_atelier_sfc), [vize_atelier_vapor](https://crates.io/crates/vize_atelier_vapor), [vize_atelier_ssr](https://crates.io/crates/vize_atelier_ssr), and [vize_patina](https://crates.io/crates/vize_patina) via [Rustler](https://github.com/rusterlium/rustler).

This crate is not meant to be used directly — it compiles automatically as part of `mix compile`.
